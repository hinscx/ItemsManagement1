商品管理系统1.0 <br>(Servlet + JSP + JDBC + MySQL)
===
     
    
     

### 功能介绍:<br>
* 对商品信息的查,增,改,删;<br>
* 可使用高级搜索根据输入的条件查询商品;<br>
* 查询的结果在分页中显示;<br>

### 运用知识:<br>
<br>
* JDBC操作数据库的基本过程;<br>
* 数据库连接池c3p0简化连接的获得;<br>
* 运用dbutil.jar包将查询到的表单数据映射到bean对象中;<br>
* JSTL标签库;<br>
* JavaWeb三层框架的思想;<br>
* 使用反射实现了请求的转发,在一个Servlet中实现了所有handler方法,体现了MVC架构的思想。<br>
